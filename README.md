

# EduPro Learner Demographics and Course Enrollment Behavior Analysis

## Project Overview

This project analyzes learner demographics and course enrollment behavior on the EduPro online learning platform.

The objective is to understand:

* Age distribution of learners
* Gender participation patterns
* Course category preferences
* Course level preferences
* Enrollment behavior across learner segments

The analysis helps EduPro make data-driven decisions for course planning, learner engagement, and platform growth.

---

## Problem Statement

EduPro wants to understand:

* Which age groups are most active
* How enrollment differs by gender
* Which course categories are most popular
* Whether learners prefer Beginner, Intermediate, or Advanced courses

This analysis provides insights into learner behavior using historical enrollment data.

## Key Insights

1. Age group 26–35 shows the highest participation.
2. Male and Female learners both actively enroll in courses.
3. Data Science is one of the most popular course categories.
4. Beginner-level courses attract a large number of learners.
5. Enrollment behavior varies across age groups and course categories.
